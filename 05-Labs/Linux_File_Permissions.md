# Linux File Permissions Investigation and Remediation

This lab documents a Linux file permission review and remediation exercise completed as part of the Google Cybersecurity Professional Certificate. The goal of the activity was to identify misconfigured permissions and apply appropriate access controls based on authorization requirements.

---

## Lab Objective
- Review file and directory permissions
- Identify unauthorized access
- Modify permissions to enforce least privilege
- Ensure users, groups, and others have appropriate access
